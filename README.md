# Emblyx Products

Official repository for Emblyx products, providing firmware releases, documentation, manuals, resources, and product-related information.

This repository serves as a central location for product documentation, firmware distribution, release notes, feature announcements, and other resources intended to support Emblyx products throughout their lifecycle.

## About

This is the official public repository for Emblyx products. It is intended to provide users, installers, integrators, and partners with access to firmware releases, documentation, product resources, and other materials related to genuine Emblyx devices.

Each product is organized in its own directory and may contain documentation, firmware, resources, and other product-specific content.

## Repository Structure

Each product has its own dedicated directory containing documentation, firmware, and supporting resources.

### Typical Structure

```text
product/
├── manuals/
├── resources/
└── firmwares/
```

The exact contents may vary between products.

## Product Availability

The contents and structure of each product directory may vary depending on the product's features, lifecycle stage, and available resources. Not all products will contain the same documentation, firmware, tools, or supporting materials.

### Manuals

The `/manuals` directory contains product-specific documentation and guidance, which may include user manuals, installation guides, technical documentation, regulatory information, and other supporting materials.

### Resources

The `/resources` directory contains supplementary product resources and reference materials. Depending on the product, this may include QR codes, configuration utilities, images, templates, product assets, and other supporting files.

### Firmwares

The `/firmwares` directory contains firmware releases, update packages, release notes, and other software components required for the installation, operation, maintenance, or upgrade of Emblyx products.

## Product Updates

New firmware releases, features, improvements, and important product announcements will be published through this repository whenever available.

Users are encouraged to periodically check for updates to ensure they are running the latest supported version of their Emblyx products.

## Contributing

We welcome bug reports, feature requests, and feedback that help improve Emblyx products.

Before opening an issue, please review our [Contributing Guidelines](.github/CONTRIBUTING.md).

When reporting a bug, please include:

* Product ID
* Hardware Revision
* Firmware Version
* Steps to Reproduce
* Relevant Logs, Screenshots, or Videos when available

Feature requests should clearly describe the problem being solved and the expected benefit.
