# Firmware Releases

⚠️ Before installing a firmware version, verify that it is compatible with your device's hardware revision.

Compatibility information is defined in `configuration.json`.

Hardware revisions sharing the same major and minor version (for example, `3.0.x`) are compatible with the same firmware versions, while revisions with different major or minor versions (for example, `3.0.x` and `3.1.x`) are not compatible.

Emblyx devices automatically receive firmware updates over the air (OTA) when new official or beta releases become available. Manual firmware installation is generally only recommended for recovery from critical software issues, troubleshooting, or testing beta firmware versions.

Always review `configuration.json` before performing a manual update.
